# virtual-pet-java
Can project <3
Virtual Pet Game (Java)

A console-based virtual pet game developed in Java to practice Object-Oriented Programming.

Features
Feed, play, and rest with your pet
Dynamic stats: satiety, energy, and happiness
Survival-based gameplay loop
Concepts Used
Encapsulation
Classes and objects
Methods and state management
Game loop logic

How to Run

Compile and run using Java:
javac *.java
java Mascotas1

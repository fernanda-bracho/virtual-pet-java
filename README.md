# Virtual Pet Game (Java OOP)
Welcome to the ultimate virtual pet simulator! This project was developed to master advanced **Object-Oriented Programming (OOP)** concepts in Java, integrating survival mechanics and resource management.

---

## Key Features
This isn't just about feeding a pet; it's about managing a delicate balance of coexistence:

* **Nature System:** Pets aren't identical. Using polymorphism, each pet inherits unique behaviors based on its "Nature":
    * 🐕**Dogs (Playful):** Will break things if ignored for too long.
    * 🐈 **Cats (Calm):** Tend to wander and get into trouble with neighbors while sleeping.
    * 🐦 **Birds (Noisy):** Test your limits with loud calls that alert the authorities.
* **Player Management:** You have stats too! If you run out of **Money** or lose your **Patience**, it's Game Over.
* **Dynamic Events:** Neighbors can complain or take drastic measures if your pet causes too much chaos.
* **Survival Loop:** Includes hunger, energy, and happiness management, plus a hygiene (bath) system that impacts overall well-being.

---

## 🛠️ OOP Concepts Applied
This project demonstrates proficiency in:
* **Abstraction:** Using abstract classes as the foundation for the `Mascota` base.
* **Interfaces:** Implementing specific behaviors through interfaces like `jugueton`, `calmado`, and `ruidoso`.
* **Polymorphism:** Dynamic event handling using `instanceof` to trigger nature-specific outcomes.
* **Encapsulation:** Protecting sensitive data via access modifiers and professional getter/setter methods.
* **State Management:** A real-time game loop that validates pet vitals and player stress levels.
